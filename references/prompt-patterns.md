# 提示词模式

## 目录

1. 纯文本生成

3. 动作与运镜参考

5. 一镜到底

7. 视频编辑

把占位符替换为实际内容。删除无关字段，不要原样输出方括号。

## 纯文本生成

```text
[比例、时长和总体风格]。[主体]位于[场景]，从[起始状态]开始，因为[触发]执行[动作因果链]，最终停在[结束状态]。摄影机以[景别和角度]开始，执行[主运动]；[光线与色彩]。[对白、环境声与音乐]。[负面约束]。
```

## 图像一致性

```text
以@图片1中的人物身份为唯一角色基准，保留[不可变特征]；服装参考@图片2，场景参考@图片3。角色从[动作起点]完成[动作序列]，情绪由[初始]变化为[结束]。摄影机[镜头设计]，光线[光线设计]。不得改变身份、服装结构、道具位置与主光方向。
```

## 动作与运镜参考

```text
主体身份参考@图片1，场景参考@图片2。只参考@视频1的[身体动作/镜头轨迹/剪辑节奏]，不参考原视频的人物身份、服装和场景。先[准备动作]，再[核心动作]，最后[稳定结束]；摄影机[主运动]，保持[连续性要求]。
```

## 广告与产品片

```text
[时长、比例]产品广告。@图片1为产品结构和包装基准，必须保持[轮廓、比例、标签、材质]。0–[A]秒：[英雄角度和环境]；[A]–[B]秒：[功能或材质展示]；[B]–[结束]秒：[产品定格和品牌信息]。灯光[主光、轮廓光、背景]，镜头[运动]。保留/禁止[Logo、文字、字幕]。
```

不要要求品牌产品“展示包装”同时又“禁止任何 Logo”。

## 一镜到底

```text
全程一镜到底，不切镜。以@图片1为起始构图，摄影机从[位置]以[运动方式]跟随[主体]，依次穿过[空间节点1]、[节点2]和[节点3]。每次场景变化通过[遮挡/门洞/光线/转身]完成，运动方向始终[方向]，最终停在@图片2对应的结束构图。保持人物身份、服装、时间和光向一致。
```

把路径控制在模型能执行的复杂度内。场景节点过多时拆段。

## 视频延长

```text
向后延长@视频1新增[X]秒。入点继承原视频最后一帧：[人物位置、动作、镜头速度、光线、声音]。新增段中，[动作发展]；摄影机继续[运动]，不得重置机位或人物姿态。结尾稳定在[尾帧状态]，为下一段/结束留出[时长]。
```

向前延长时描述原视频第一帧之前的状态，并让结尾准确进入原视频首帧。

## 视频编辑

```text
编辑@视频1。保留[镜头、动作、节奏、场景、声音]；只把[目标元素]替换为@图片1中的[对象]。替换后必须保持原有遮挡、比例、光向、接触关系和动作时序；其他元素不得变化。
```

一次编辑尽量只处理一类变化。

## 音乐卡点

```text
画面素材使用@图片1至@图片N，音乐和节拍参考@音频1。按[节拍/小节/鼓点]切换，每次切换对应[构图变化或动作峰值]；在[时间点]进入高潮，在[时间点]留出品牌或尾帧。允许改变景别，但保持[人物/产品/色彩]一致。
```

没有实际音频分析结果时，不虚构 BPM 或精确鼓点。

## 多段项目

为每段使用同一模板：

```text
片段N：[时长]
入点：[继承状态]
剧情目标：[本段唯一目标]
动作：[起点→触发→发展→结果]
摄影：[景别、角度、主运动]
声音：[对白、环境、音乐]
出点：[可见尾帧状态]
接力锚点：[下一段必须复述的元素]
```

先生成锚点最清楚的片段，再生成依赖其尾帧的片段。
