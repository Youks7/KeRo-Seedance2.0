# 评审与故障诊断

## 交付前评分

每项 0–2 分：0 缺失，1 模糊，2 明确可执行。总分低于 16 时先修改再交付。

| 项目 | 检查问题 |
|---|---|
| 目标 | 这段视频只承担一个清晰目标吗？ |
| 主体 | 身份、外观和不可变项明确吗？ |
| 素材 | 每个 `@引用` 都存在且职责明确吗？ |
| 动作 | 有起点、触发、发展和结果吗？ |
| 摄影 | 景别、角度和主运动是否兼容？ |
| 空间 | 位置、方向、轴线和遮挡是否清楚？ |
| 光色 | 光向、时间、天气和色彩逻辑一致吗？ |
| 时间 | 时间段连续且不超出目标时长吗？ |
| 声音 | 说话者、声音层和同步节点明确吗？ |
| 结尾 | 有稳定尾帧或明确接力状态吗？ |

## 常见失败与修复

### 角色漂移

只保留一个身份基准；重复不可变特征；减少换装、遮挡和大幅转身；重新入画时使用上一段尾帧。

### 肢体或多人融合

给每人独立位置和动作顺序；把接触拆成准备、接触、分离；减少背景人物交叉；先用中景或全景保证完整动作。

### 镜头失控

保留一个主运动；写出镜头起点、路径和终点；给运动添加叙事目标；复杂路线拆段。

### 动作缺乏因果

增加可见或可听触发，说明动作改变了什么，使用连续动词而不是形容词堆叠。

### 转场跳变

复述上一段出点；使用遮挡、失焦、黑场或动作匹配；保持镜头速度和方向；用声音桥连接。

### 产品变形

明确不可变轮廓和标签位置；一次只做一种特效；先生成静态英雄镜头；使用多角度参考图。

### 文字不可读

把文字放在稳定尾镜；减少字数；给出准确内容和位置；关键文字优先在后期合成。

### 音画不同步

缩短对白；标记同步动作；使用真实音频参考；将复杂对话拆镜。

## 迭代记录

记录修改层、预期问题、实际改善、新增问题和是否保留版本。用结果反馈改写下一条提示词，不要无差别增加关键词。
