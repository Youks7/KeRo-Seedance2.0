# 电影视觉语法索引

把电影当作摄影问题的解决方案库。先判断当前镜头要解决的问题，再提取构图、光线、运动、节奏或声音原则；不要只写片名，也不要复刻具体角色、台词和受保护画面。

## 使用方法

1. 选择一部主参考解决核心问题；
2. 最多增加一部次参考补充另一维度；
3. 把片名转换成可观察属性；
4. 删除片名后，提示词仍应能表达完整视觉意图。

## 转换示例

不要只写：

```text
像《花样年华》一样。
```

改写为：

```text
狭窄走廊中的框中框构图，暖色壁灯与冷色窗光分割人物，镜头隔着门框观察，动作克制，步速放慢，衣料和环境声清晰，画面带轻微胶片颗粒与错过感。
```
