# 电影视觉模式索引

把经典电影当作摄影问题的解决方案库。只提取可迁移的构图、光线、镜头、节奏和叙事原则；不要复制具体镜头、角色、台词或受保护画面。

| 影片 | 可迁移模式 | 适用任务 |
|---|---|---|
| 《1917》 | 隐形接点、跟随长镜头、空间连续性 | 一镜到底、沉浸式行动 |
| 《2001太空漫游》 | 中心对称、深焦、极简未来主义、缓慢节奏 | 科技、建筑、宏大秩序 |
| 《阿拉伯的劳伦斯》 | 极远景、人物尺度对比、热浪与地平线 | 沙漠、史诗、孤独 |
| 《巴里·林登》 | 自然光与烛光、绘画式静态构图、缓慢变焦 | 古典、奢侈品、历史 |
| 《盗梦空间》 | 建筑空间变形、多层时空、实用特效感 | 概念片、梦境、转场 |
| 《疯狂的麦克斯：狂暴之路》 | 中心动作、清晰方向、暖沙冷天色、快速剪辑 | 动作、车辆、卡点 |

| 《荒野猎人》 | 自然光、广角近身、长镜头、恶劣天气 | 生存、自然、身体感 |

## 选择方式

- 需要空间连续：参考《1917》；
- 需要宏大尺度：参考《阿拉伯的劳伦斯》《沙丘》；
- 需要人物权力：参考《教父》《公民凯恩》；
- 需要情绪亲密：参考《花样年华》；
- 需要动作清晰：参考《疯狂的麦克斯：狂暴之路》；
- 需要动画视觉规则：参考《千与千寻》《蜘蛛侠：平行宇宙》；
- 需要赛博夜景：参考《银翼杀手》系列。

把选择结果转换成独立属性，不要只输出片名。
