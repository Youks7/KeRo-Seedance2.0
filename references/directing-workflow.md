## 视觉宇宙

先建立一页视觉圣经，再写镜头。至少固定世界类型与时代、天气和时间、主色与禁用色、主光方向和对比度、材质、摄影机高度与运动方式、画面密度与留白。

把风格写成可观察属性，而不是只写“电影感”“高级感”。例如：

```text
低饱和青灰环境，暖色实用灯作为局部色彩锚点；阴天软光，人物面部保持半级欠曝；中长焦压缩空间，镜头运动克制。
```

## 角色锚点

为每个主要角色建立独立锚点：

```text
角色A：30岁左右，短卷发，左眉尾有小痣，瘦高体型；深蓝短夹克、灰色圆领衫、旧帆布包；说话前习惯短暂停顿，紧张时右手捏包带。
```

区分四层：

1. 身份不可变项；
2. 当前场景服装；
3. 当前情绪与表演；
4. 当前镜头动作。

不要把临时情绪写入永久身份，也不要把镜头动作写成角色外貌。

### 三类参考图

条件允许时分别准备：

- 身份图：正面或三分之二角度，光线中性；
- 服装图：能看清全身比例和材质；
- 道具图：能看清结构、尺度和磨损。

素材不足时，优先保证身份图，不要用风格化过强或严重遮挡的图片承担身份锚定。

## 产品锚点

固定外轮廓和比例、材质与反射、标签与接口、品牌文字、允许变形部位、展示顺序和英雄角度。

对品牌包装不要默认禁止 Logo。需要无品牌版本时，明确要求移除或替换。

## 镜头连续性

跨镜头记录以下接力状态：

| 类别 | 接力内容 |
|---|---|
| 人物 | 位置、朝向、视线、手势、情绪强度 |
| 道具 | 所在手、开合状态、损伤、相对位置 |
| 摄影 | 景别、轴线、运动方向、镜头高度 |
| 环境 | 光向、天气、背景人流、烟尘和水面状态 |
| 声音 | 音乐小节、环境声连续性、对白尾音 |

下一段的第一句应复述上一段的关键出点状态，而不是只说“接着上一段”。

## 多角色调度

为每个角色编号并分别描述：

```text
角色A位于画面左前景，面向角色B；角色B位于右侧中景，先后退一步再抬头；背景群众只沿纵深方向移动，不与两位主角发生身体接触。
```

遵守：

- 给角色不同的空间区域、服装色或动作节奏；
- 明确谁先动作、谁反应；
- 一次只安排一个关键交互；
- 复杂互动拆成准备、接触、结果三个阶段；
- 用眼神、道具和空间关系建立因果。

## 表演与参考视频

把参考视频拆成身体动作、表情与视线、镜头轨迹、剪辑节奏、声音节拍等层，并声明只参考需要的层。例如：

```text
参考@视频1的肩部动作节奏和停顿时机；不参考其中人物身份、服装、场景与色彩。
```

对精细表演写动作动机：角色看见什么、为何反应、反应从何种强度变化到何种强度。

## 生成顺序

1. 先测试角色和产品静态一致性；
2. 再测试单一动作；
3. 再加入主镜头运动；
4. 再加入复杂环境和次要角色；
5. 最后处理对白、文字和声音同步；
6. 锁定可用尾帧后再生成续段。

不要先生成最复杂的高潮镜头。先用低风险镜头确认视觉圣经和锚点有效。
