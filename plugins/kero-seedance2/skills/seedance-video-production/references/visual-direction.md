# 视觉导演与镜头语言

在用户需要建立完整视觉风格、选择镜头语言、设计天气光影或把情绪转成画面时读取。不要为只要一条简单提示词的任务整份照搬。

## 视觉圣经

先把抽象风格翻译成可观察属性：

| 维度 | 要决定的内容 |
|---|---|
| 色彩 | 主色、点缀色、饱和度、镜头间过渡 |
| 光线 | 时间、方向、软硬、明暗比、实际光源 |
| 构图 | 主体位置、负空间、前中后景、对称或失衡 |
| 光学 | 广角/长焦/微距、景深、变形宽银幕、颗粒 |
| 材质 | 新旧、磨损、灰尘、织物、金属、皮肤和环境反应 |
| 运动 | 摄影机速度、角色动作幅度、整体节奏 |
| 情绪 | 压迫、孤独、诗意、危险、权力、亲密等主要感受 |

选择少量高价值训练语义锚点，例如 `National Geographic`、`IMAX`、`35mm film grain`、`documentary realism`、`240fps super slow motion`。让它们与具体画面属性协同，不用同义词反复堆叠。

需要跨镜头统一时，先写一段视觉圣经，再为每个镜头只补充变化项。不要把整段视觉圣经机械复制十次。

## 镜头构造公式

按任务需要从下列层中选择：

```text
景别 + 角度 + 一个主运动 → 主体与可见动作 → 场景与环境反应
→ 光线与色彩 → 构图或光学 → 质感与情绪 → 稳定结束状态
```

不是每个镜头都必须填满全部字段。短镜头优先保证：

1. 主体和动作一眼可读；
2. 镜头起点、路径和终点兼容；
3. 光线与环境服从同一物理逻辑；
4. 结尾可用于停帧、剪辑或续段。

## 情绪到镜头的映射

把下表当作候选组合，不是固定咒语：

| 目标感受 | 可选视觉语法 |
|---|---|
| 史诗 | 极远景、低机位、缓慢升起、巨大负空间、人物尺度对比 |
| 紧张 | 特写、轻微倾斜、缓慢逼近、低调光、局部遮挡 |
| 孤独 | 远景、固定镜头、大量负空间、低饱和、自然环境声 |
| 危险 | 中近景、侧逆光、深阴影、浅景深、有限手持 |
| 诗意 | 中景或远景、侧面、柔光、类似色、静止或极慢运动 |
| 权力 | 低机位、顶光或明暗对比、深焦、对称、克制运动 |
| 心理崩裂 | 大特写、失焦转清晰、镜像、轻微荷兰角、抽帧或跳切 |
| 浪漫 | 中近景、逆光、黄金/蓝调时刻、浅景深、柔和暖光 |
| 纪实 | 观察式手持、可用光、低饱和、环境完整、不过度美化 |
| 速度 | 侧面跟拍、方向清楚、主体中心稳定、背景运动模糊、速度变化有节点 |

组合时检查冲突。例如，不能同时要求极浅景深和前中后景全部清晰，也不要把固定镜头、快速环绕和高速推镜塞进同一短镜头。

## 光线、色彩与天气

### 色彩签名

固定一个主色系统和一个小面积点缀色。跨镜头改变时间或空间时，设计中间色或过渡光源，不让冷暖关系无理由翻转。

### 时间光线

- 黄金时刻：低角度暖光、长阴影、金色轮廓，适合希望、怀旧和亲密。
- 蓝调时刻：深蓝环境、暖色实际光源，适合孤独、神秘和未来感。
- 阴天：大面积柔光、低对比，适合写实、冷静和压抑。
- 夜景：明确主要实际光源及其方向，不只写“电影夜景”。

### 天气必须产生环境反应

| 天气 | 同时描述 |
|---|---|
| 雨 | 强度与方向、湿表面、积水/飞溅、衣物吸水、反射光、雨声 |
| 雪 | 密度、风向、落地积累、脚印/扬雪、冷漫射光、能见度 |
| 雾 | 前中后景浓度、主体显隐、体积光、轮廓衰减、移动方向 |
| 火 | 火源类型、闪烁、烟和余烬、对人物与环境的暖光和阴影 |
| 沙尘 | 风向、尘墙尺度、前景碎屑、天空变色、主体受力与遮挡 |

不要只写“下大雨”而不给地面、服装、光线和声音任何反应。

## 复杂运镜

复杂镜头必须服务于揭示信息、建立空间或改变情绪。

- 一镜到底：写连续路径和空间节点，限制角色与道具变化；过长时用遮挡、黑场、失焦或相似运动拼成伪长镜头。
- 环绕：写方向、角度范围、半径、速度和主体是否转身；短片优先 90–180 度，不默认 360 度。
- 变速：写正常速度、升格节点和恢复常速节点，让慢动作落在决定性动作上。
- 焦点转移：写从什么对象转到什么对象，以及触发转移的事件。
- 手持：说明是轻微纪录感还是剧烈冲击，不把“手持”当作随机抖动。

## 交付检查

- 风格词是否已转成可见属性；
- 镜头是否只有一个主运动；
- 构图、景深和焦段是否冲突；
- 天气是否影响了环境、主体、光线和声音；
- 复杂运镜是否有清楚路径与叙事目的；
- 是否保留了用户明确要求的训练语义锚点。
